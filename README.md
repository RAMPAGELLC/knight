## üöÄ Getting Started

You can start using Knight in three ways:

### Option 1: Clone via Git

```bash
git clone https://github.com/RAMPAGELLC/knight.git
```

### Option 2: Using Wally

Add `knight = "vq9o/knight@latest"` to `wally.toml` then run `wally install`.
