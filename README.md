## üöÄ Getting Started

You can start using Knight in two ways:

### Option 1: Clone via Git
```bash
git clone https://github.com/RAMPAGELLC/knight.git
```

### Option 2: Use Knight CLI
If you have Knight CLI installed:
```bash
knight init
```
